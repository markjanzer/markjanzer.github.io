*What did you learn about CSS positioning?*

I learned that there are many ways to position elements in the same way. I can use floats or relative positioning, or I can place elements inside of others, or I can use absolute positioning. The thing that I want to learn more about is best practice. I want to know whether to use relative positioning or inline-blocks or floats to achieve an element placement.

*What did you learn about adding and formatting elements with CSS in this challenge?*

I learned that although I really like the atomic method, I don't think that it is always the best way to go about things. There is a lot of margin spacing and padding that I only would ever use once on one title or picture. These titles and pictures had a good amount of unique CSS on them, so I prefered to have larger blocks of CSS for a class that is only used once. Below these blocks, I used atomic with great success.

I also found that it is very easy to spend large amounts of time in CSS.
