Intro on Command Line, Git, and GitHub

September 11th, 2015

Command line

In this post, I am going to introduce Command Line, Git, and GitHub.First I will go over the basic concepts, and then I will clarify some misconceptions that I had going into DBC.

Command line is our access to a computer's raw computing power. The first computers were designed without monitors, everything was done through command line. Most things (if not everything) that you do on your computer can be done through Command Line, it just doesn't have a convinient user interface (such as the files you can drag around to different windows).
